# Конфигурационное управление

## 1. Общее описание

Данный проект представляет собой инструмент командной строки для обработки учебного
конфигурационного языка. Инструмент выполняет синтаксический анализ входного файла,
выявляет синтаксические ошибки и вычисляет значения констант и словарей.

Синтаксический разбор реализован с использованием специализированного инструмента
**Lark** (LALR-парсер). Входной текст читается из файла, путь к которому передаётся
через аргументы командной строки. Результат трансляции выводится в стандартный поток
вывода.

Проект разработан в рамках домашней работы.

## 2. Описание функций и настроек

### Поддерживаемые конструкции языка

#### Комментарии

Однострочные комментарии:
```
-- Это однострочный комментарий
```

Многострочные комментарии:
```
{!
Многострочный
комментарий
!}
```

#### Числа

Двоичные числа:
```
0b10101
```

#### Строки

```
'Это строка'
```

#### Имена

```
[a-zA-Z][a-zA-Z0-9]*
```

#### Словари

```
[
  имя => значение,
  имя => значение
]
```

#### Объявление констант

```
имя is значение
```

#### Использование констант

```
${имя}
```

---

### Архитектура проекта

- `grammar.lark` — описание грамматики языка.
- `transformer.py` — вычисление значений и обработка конструкций.
- `main.py` — интерфейс командной строки.
- `examples/` — примеры входных файлов.

## 3. Сборка проекта и запуск тестов

### Требования

- Python 3.10+
- pip

### Установка зависимостей

```
pip install lark
```

### Запуск программы

```
py main.py <путь_к_файлу>
```

Пример:
```
py main.py test.cfg
```

## 4. Примеры использования

### Пример 1

Вход:
```
a is 0b101
b is 'hello'
```

Выход:
```
a = 5
b = hello
```

### Пример 2

Вход:
```
a is 0b101
b is 'hello'

dict is [
  x => ${a},
  y => ${b}
]
```

Выход:
```
a = 5
b = hello
dict = {'x': 5, 'y': 'hello'}
```
